Ejercicio Real con Contratos

Para poder jugar con filter y map se se puede pedir que:

  1. Obtengan todos los contratos cuyo campo TIPPRODUCT tenga un valor determinado, por ejemplo “KT” (filter)
  2. Obtengan un nuevo array en el que cada elemento contenga el código de contrato concatenado con el digito de control (campos codCONTRAT y digContrat), además del saldo en euros (campo SALCONTRAT dividido entre 100) (map)
  3. Obtengan todos los contratos que en su campo acciones contengan la clave “SITUACION” (filter por la subestructura)
  4. Obtengan un array que contenga todas las acciones posibles (map y spread)
  5. Combinaciones de las anteriores, etc

En todo lo anterior no pueden usar bucles for ni foreach, por supuesto :-)

Filers, Map and Reduce

Filer, Map, Reduce

Array.filter – keep all that match a condition.


Ejemplo
var words ['verde', 'marron', 'rojo']; const result = words.filter( word => word.length>4 ))
NOTE: there is a small mistake in the animation. It should return [6,7], not [6,7,8]. I’ll fix it shortly. . .
  • function value > 5 is applied to every item in the original array.
  • .filter() returns a modified copy of the array – the original is still available!
  • Result: [6,7,8] (only values that passed the condition.)

Array.map – map all values to an expression.


Ejemplo
  • Expression value + 1 is applied to every item in the original array.
  • .map() returns a modified copy of the array leaving original untouched.
  • Result: [2,3,4,5,6,7,8] (a copy of the original array is created.)

Array.reduce – reduce all items to a single value.


Ejemplo
  • Reducer function F takes value and accumulator.
  • In this example .reduce(v, a) returns the sum of all values.
  • Result: 28 (the sum of all numbers in the original array)
